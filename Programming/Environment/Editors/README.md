# My Status

## 2023-06-02:

I've been using Sublime Text for a little over a decade now.
I've been using Vim competently (not elegantly) for server-side and little tasks for almost as long.
I have outgrown these.

As for Sublime, it's proprietary-ness has always annoyed me, and now it's nagging me for a "Key Upgrade".
So even though I paid for it, it's become nagware again; that was fine when I was 8 and playing video games,
  but I have standards now.
Also, while it is configurable, the documentation leaves something to be desired.
Perhaps there are gaps in the API as well, but I wouldn't know: the docs aren't especially discoverable.

As for Vim, it's a great editor!
The only complaint I have during day-to-day use is that (the way I use it) it's completely terminal-based.
Terminals have kinda broken keyboard input, and there's nothing a TUI Vim can do about that.
